# Solution

Place the exported **unmanaged** Power Platform solution `.zip` in this folder.

- Download the unmanaged solution into a **development environment**, then update
  it as needed (adjust the plugin step, environment variable, etc.).
- Re-export the updated unmanaged solution here to keep source control current.
- Deploy to downstream environments as a **managed** solution using your normal
  ALM process (pipeline, `pac solution import`, or managed export/import).
